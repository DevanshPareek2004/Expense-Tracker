🚀 ExpenseFlow – Smart Expense Tracker 💰📊
ExpenseFlow is a powerful and user-friendly expense tracking application designed to help users manage their finances efficiently. It categorizes expenses, provides insightful visualizations, and generates detailed reports.

🔥 Features
✅ Expense Tracking – Log and categorize expenses easily
✅ Data Visualization – Graphical insights using Matplotlib & Seaborn 📊
✅ PDF Reports – Generate downloadable reports with fpdf 📝
✅ Email Notifications – Get spending alerts via SMTP 📩
✅ Interactive Dashboard – Built with HTML, CSS, JavaScript, Flask for a seamless UI

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Flask
Libraries: Pandas, Matplotlib, Seaborn, fpdf, smtplib
